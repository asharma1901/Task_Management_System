
#  Task Management System (Spring Boot)

A simple RESTful Task Management System built using **Spring Boot**, supporting task creation, retrieval, updating, deletion, pagination, and validation.

---

##  Features

- Create, Read, Update, Delete (CRUD) tasks
- Pagination support for listing tasks
- Proper HTTP status codes (`201`, `200`, `204`, `404`, `400`)
- Input validation using Bean Validation
- Future date validation for task due dates
- Clean REST API design

---

##  Tech Stack

- **Java 17+**
- **Spring Boot**
- **Spring Web**
- **Spring Validation (Jakarta Validation)**
- **JUnit 5 & Mockito** (for testing)
- **Maven** (build tool)

---


##  Prerequisites

Ensure the following are installed on your system:

- **Java JDK 17 or higher**

- Maven 3.8+

---

## Clone the Repository
git clone https://github.com/asharma1901/Task_Management_System.git
cd task-management-system

---

## Install Dependencies

All dependencies are managed using Maven.

Run the following command to download and install dependencies:

mvn clean install

---

### This will:

- Download all required libraries
- Compile the project
- Run unit tests

---

## Run the Application
mvn spring-boot:run

The application will start on:

http://localhost:8080
